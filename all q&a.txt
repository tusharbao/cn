1. Demonstrate different types of topologies and transmission media
Q1. What is network topology?
Hindi-English: Topology ka matlab hota hai network me devices ka connection pattern.
English: Network topology is the physical or logical layout of connected devices.
________________________________________
Q2. Name different types of network topologies.
Hindi-English: Bus, Star, Ring, Mesh, Tree, Hybrid.
English: Bus, Star, Ring, Mesh, Tree, and Hybrid topologies.
________________________________________
Q3. Difference between Bus, Star, and Ring topology?
Hindi-English:
•	Bus: Single cable sabko connect karti hai.
•	Star: Sab devices ek central switch/hub se connect hote hain.
•	Ring: Devices ek loop (ring) me connected hote hain.
English:
Bus uses a single backbone cable, Star connects all devices to a central device, Ring connects devices in a closed loop.
________________________________________
Q4. What are transmission media?
Hindi-English: Data transfer ke liye use hone wale path ko transmission media kehte hain.
English: Transmission media are the physical paths for data transmission.
________________________________________
Q5. Difference between guided and unguided media?
Hindi-English:
•	Guided: Wired media (cables) jaise Twisted Pair, Coaxial, Fiber.
•	Unguided: Wireless media jaise Radio, Microwave, Infrared.
English:
Guided media use cables; unguided media use air for wireless communication.
________________________________________
Q6. Which transmission medium is used in wired LAN?
Hindi-English: Twisted Pair Cable (Cat5e / Cat6).
English: Twisted pair cables are used in wired LANs.
________________________________________
Q7. What tool is used to simulate networks?
Hindi-English: Cisco Packet Tracer.
English: Cisco Packet Tracer.
________________________________________
Q8. What are the uses of Packet Tracer?
Hindi-English: Network design, simulation, configuration aur testing ke liye.
English: Used for designing, simulating, configuring, and testing networks.
________________________________________
Q9. How can you check connectivity between nodes?
Hindi-English: PING command use karke.
English: Using the PING command.
________________________________________
Q10. What happens when you change the topology type?
Hindi-English: Data flow path aur connectivity change ho jaati hai.
English: Changing topology changes data flow and network connections.

6. UDP Socket Program for File Transfer
Q1. What is UDP?
Hindi-English: UDP (User Datagram Protocol) ek fast, connectionless protocol hai.
English: UDP is a fast, connectionless protocol.
________________________________________
Q2. How is UDP different from TCP?
Hindi-English: UDP me connection nahi banta aur reliability kam hoti hai.
English: UDP doesn’t establish a connection and is less reliable than TCP.
________________________________________
Q3. What is meant by connectionless service?
Hindi-English: Data bina connection ke bheja jata hai.
English: Data is sent without establishing a connection.
________________________________________
Q4. Is UDP reliable? Why or why not?
Hindi-English: Nahi, kyunki error checking aur acknowledgment nahi hota.
English: No, because it lacks acknowledgment and error recovery.
________________________________________
Q5. Real-life examples of UDP?
Hindi-English: Video streaming, VoIP, online gaming.
English: Used in video streaming, VoIP, gaming.
________________________________________
Q6. What functions are used for UDP communication?
Hindi-English: socket(), sendto(), recvfrom(), bind().
English: socket(), sendto(), recvfrom(), bind().
________________________________________
Q7. How do you send and receive files using UDP?
Hindi-English: Client file ko chunks me send karta hai aur server recvfrom() se receive karta hai.
English: Client sends file chunks; server receives with recvfrom().
________________________________________
Q8. What types of files can be transferred?
Hindi-English: Text, image, or any binary file.
English: Text, image, or any binary files.
________________________________________
Q9. What happens if a packet is lost in UDP?
Hindi-English: Packet resend nahi hota; data loss hota hai.
English: Lost packets are not retransmitted, causing data loss.
________________________________________
Q10. Why is UDP preferred for audio/video transfer?
Hindi-English: Kyunki fast hai aur thoda packet loss acceptable hai.
English: It’s faster, and small packet loss doesn’t affect real-time media.

11. Installing and Configuring DHCP Server
Q1. What is DHCP?
Hindi-English: DHCP (Dynamic Host Configuration Protocol) IP addresses automatically assign karta hai.
English: DHCP automatically assigns IP addresses to devices.
________________________________________
Q2. What is the function of a DHCP server?
Hindi-English: Clients ko automatic IP, subnet mask, gateway deta hai.
English: Assigns IP, subnet mask, and gateway to clients.
________________________________________
Q3. What are the advantages of DHCP?
Hindi-English: Time saving, less manual work, aur IP conflicts kam.
English: Saves time, reduces errors, and avoids IP conflicts.
________________________________________
Q4. Difference between static and dynamic IP assignment?
Hindi-English:
•	Static: Manually set hoti hai
•	Dynamic: DHCP se auto milti hai
English:
Static is manual; dynamic is automatic via DHCP.
________________________________________
Q5. What is a DHCP lease?
Hindi-English: Temporary period jisme client ko IP assign hota hai.
English: The time duration for which an IP is assigned to a client.
________________________________________
Q6. Steps of DHCP process (DORA)?
Hindi-English: Discover → Offer → Request → Acknowledge.
English: Discover, Offer, Request, Acknowledge.
________________________________________
Q7. How do you configure a DHCP server in Packet Tracer?
Hindi-English: Server → Services → DHCP → Pool set karo (Network, Mask, Gateway, DNS).
English: In Packet Tracer, configure DHCP service by setting pool details.
________________________________________
Q8. How do clients obtain IP addresses automatically?
Hindi-English: DHCP enable karke “Obtain IP Automatically” option select karte hain.
English: Clients get IPs automatically when DHCP is enabled.
________________________________________
Q9. What command is used to check assigned IP?
Hindi-English: Windows me ipconfig command.
English: Use ipconfig to check assigned IP.
________________________________________
Q10. What happens when DHCP server fails?
Hindi-English: Clients ko IP nahi milta, network disconnect ho jata hai.
English: Clients won’t receive IPs, leading to network failure.
